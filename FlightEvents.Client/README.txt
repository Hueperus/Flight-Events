If you are running this for the first time, you might get a prompt to install .NET Core runtime.

You can click yes to get directed to https://dotnet.microsoft.com/download/dotnet-core/3.1
You can then download and isntall "Desktop Runtime 3.1.3" x86 (even on 64-bit OS).

Alternatively, you can download the installer directly at https://dotnet.microsoft.com/download/dotnet-core/thank-you/runtime-desktop-3.1.3-windows-x86-installer

You will also need SimConnect, which is bundled with FSX or FSX Steam Edition or can be downloaded from https://events-storage.flighttracker.tech/downloads/SimConnect.zip and installed according to the ReadMe.txt in the download zip.
